## Python Basic ML Classifier Algorithms
- Perceptron
- Adaline w/ Gradient Descent
- Adaline w/ Stochastic Gradient Descent

## Usage
- `pip install -r requirements.txt`
- From any sub algorithm folder, `python ./main.py`
